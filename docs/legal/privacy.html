Your privacy is of utmost importance to the Zenvix Development Group. This Privacy Policy governs how Zenvix handles developer information on our local environment platform.

1. Zero Data Collection Architecture

Zenvix operates entirely as a local-first system. All configurations, project database files, paths, and server states are stored directly on your physical drive (typically under C:\Zenvix). We do not transmit, sync, or replicate your local code repositories or database structures to external cloud environments.

2. Development Telemetry

Zenvix does not include tracking systems or usage monitors. The package works completely offline, with no persistent network connection required. Update notifications are checked manually or via official secure GitHub release endpoints.

3. Third-Party Web Services

When running local servers (e.g., Apache, Nginx, PHP, or phpMyAdmin), all traffic is restricted strictly to your local machine (127.0.0.1 / localhost). Any port exposure, firewall rules, or DNS proxies are managed purely within your local operating system constraints.

4. Updates & Modifications

We reserve the right to modify this local privacy commitment as the tool expands. Updates are always delivered transparently via our repository.
