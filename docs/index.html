Setup Walkthrough

Get ZENVIX up and running on your local machine in three quick steps.

1

Run the Wizard Installer

Download the standalone installer and run the wizard. It sets up files in a secured C:\Zenvix directory, creates desktop icons, and configures Windows Services automatically.

2

Launch and Start Services

Open the app control panel. Start NGINX/Apache, PHP, and MySQL from the UI. Runtimes will spin up instantly in secure, isolated sandbox environments.

3

Create Projects & Databases

Access phpMyAdmin directly at localhost:8080/phpmyadmin/ to create tables. Map directories to automatic local domains instantly.

Windows System Requirements

ZENVIX is built from the ground up strictly using native Win32 controls, NGINX core binaries, and .NET 8.0, guaranteeing extremely fast performance and near-zero memory footprint.

  • OS Version: Windows 10 or Windows 11 (64-bit strictly supported)
  • Memory: 4GB RAM minimum (runs lightweight on less than 150MB total!)
  • Storage: 250MB free space (bundles PHP 8.3, Apache, Nginx, SQLite & databases)
  • Permissions: Administrator privileges are recommended during installation to auto-configure hosts files and Windows services.
